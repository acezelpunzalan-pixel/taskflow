📥 Export Data

Download a snapshot of everything currently in TaskFlow (sites, vendors, users, roles, tasks, field visits, invoices, devices, CapEx, OpEx, and workflows) as one JSON file. Keep this file in your repo alongside the code as a backup, or use it to move data between browsers/devices.

📤 Import Data

Restore or populate data from a previously exported taskflow-data.json file. Records are matched by ID — importing will update existing records and add new ones, it will not delete anything that's missing from the file.
